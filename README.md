# Calculator
A simple console-based calculator written in Java. Supports basic arithmetic operations (+, -, *, /) with input validation and error handling.

# 🧮 Java Console Calculator

Простой консольный калькулятор на Java, который умеет выполнять базовые арифметические операции и проверяет корректность ввода.

---

## 🚀 Возможности

* Поддержка операций:

  * Сложение (`+`)
  * Вычитание (`-`)
  * Умножение (`*`)
  * Деление (`/`)
* Проверка корректности ввода
* Обработка ошибок:

  * Неверный формат выражения
  * Деление на ноль
* Красивый вывод результата (без `.0` для целых чисел)
* Бесконечный цикл работы (до команды `exit`)

---

## 📥 Пример использования

```
Введите выражение: 10+5
Результат: 15

Введите выражение: 7/2
Результат: 3.5

Введите выражение: 10/0
Ошибка: деление на ноль невозможно.

Введите выражение: 5++2
Ошибка: неверный формат ввода.
```

---

## 🛠️ Как запустить

1. Убедитесь, что у вас установлен Java (JDK)
2. Скомпилируйте программу:

```
javac Calculator.java
```

3. Запустите:

```
java Calculator
```

---

## ⌨️ Формат ввода

* Вводите выражение **без пробелов**, например:

  ```
  12+34
  7*8
  100/5
  ```
* Для выхода введите:

  ```
  exit
  ```

---

## ⚠️ Ограничения

* Поддерживается только **одно арифметическое действие за раз**
* Нет поддержки:

  * Скобок
  * Дробных чисел на входе (только целые)
  * Сложных выражений (например `2+3*4`)
